# Prim 算法生成迷宫

  1. 初始地图所有位置均设为墙。
  2. 选一个单元格作为迷宫的通路，即起点，然后把它的邻墙放入列表。
  3. 当列表里还有墙时，从列表里随机选一个墙：
    1. 若这面墙上下左右四个位置只有一个位置是路，则从列表中移除这面墙，并将其设置为路，同时将其周围（上下左右）的所有墙放入列表。
    2. 否则从列表中移除这面墙。
  4. 当列表为空时结束。

# A * 算法自动寻路

定义节点 n 的代价为 f (n) = g (n) + h (n)，并以此建立小根堆。其中 g (n) 为该点到起点的真实距离，h (n) 为该点到终点的估计距离。估计距离为曼哈顿距离。
以下为 A * 算法具体步骤：

  1. 将起点插入小根堆。
  2. 当堆不空时，取出堆顶元素 t：
    1. 若 t 为终点，则结束搜索，并回溯记录路径。
    2. 否则遍历 t 周围为路的节点，更新距离并插入堆中。
