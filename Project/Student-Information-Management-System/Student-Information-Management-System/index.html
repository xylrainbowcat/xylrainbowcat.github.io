# 团队分工

本项目是一个团队项目，由徐永乐，黄天权，吴韦烨，贾常鸣共同完成。

其中徐永乐完成了需求分析、代码（50%）和功能测试占项目的 30%；
吴韦烨完成了代码（50%）和 GUI 界面占项目的 30%；
黄天权完成了数据库的建立、部分数据集占项目的 20%；
贾常鸣完成了 PPT 和数据集占项目的 20%。

代码部分详细分工：
徐永乐：系统登录界面、学生端成绩查询、教师端成绩录入
吴韦烨：学生端选课中心、管理员端师生管理、管理员端课程管理
