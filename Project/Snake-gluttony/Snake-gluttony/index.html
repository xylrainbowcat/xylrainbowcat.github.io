# AI 贪吃蛇自动寻路

由于蛇头永远在占领空格，而蛇尾永远在留出空格，因此可以规定：若蛇头能找到一条路径到蛇尾，那么此时蛇是安全的。基于此可以制定以下策略：

  1. 若蛇可以找到吃食物的路径，则派出一条 “虚拟蛇” 沿最短路去吃食物。
    1. 若 “虚拟蛇” 吃完食物是安全的（蛇头能找到蛇尾），就派真蛇沿最短路去吃食物。
    2. 否则，真蛇不吃食物，沿最长路跟着尾巴走。
  2. 若蛇找不到吃食物的路径，则沿最长路跟着尾巴走。

由于地图不大，这里找最短路和最长路均使用 bfs 实现，找最长路的策略是每一步都沿三个方向的最短路中最长的一条走。
